# User Manual

Restaurant and Hybrid operators should also follow [Restaurant Operations](RESTAURANT_OPERATIONS.md) for tables, held orders, employee edit locks, item notes, and checkout.

WholesalePOS will support daily store workflows across POS checkout, inventory, customers, suppliers, reports, and administration.

Current milestone provides the application shell and prepares the system for authenticated setup.

In Restaurant or Hybrid mode, every active product from the existing POS catalog is available. Confirm an order to reserve stock; the selector and POS then show the quantity still available after active reservations.

Use Order recovery to undo a saved item mistake, merge active orders, or split selected quantities into a separate bill. Each recovery action requires a reason. Managers can show and restore inactive tables. Authorized refund and void controls appear on completed Restaurant orders and confirm that inventory will be restored.
